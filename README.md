# Snail - Email

Built to be compatible with ALL email hosts(within reason), and selfhosted/deployed as easiliy and cheaply as possible, and to be compatible with as many email clients as possible to allow you to use the email in way that is productive and non-invasive, it also has a fully featured plugin integration service that allows it to be as extendable as your coding ablities allow.

The goal being to make it possible for people to own their email without some big corporation knowing every private detail of your life, supports custom domains.

No build in @{{company name}}.com so domain is required.

Built with Rust and Typescript!
